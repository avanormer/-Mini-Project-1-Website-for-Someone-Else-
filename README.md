# Mini Project 1: Website for The Service Benefit

## Client
Deniz Ketenci, Director of The Service Benefit

## Project Description
This website is for The Service Benefit (TSB), a platform that highlights impactful non-governmental organizations (NGOs), promotes volunteer opportunities, and encourages donations to support global causes. The site features a clean, accessible design with pages for the home, about, NGOs, and contact information.

 ## What this is
A simple website that highlights The Service Benefit’s mission and showcases Filipino NGOs through media and calls to action (donate/volunteer/share).
