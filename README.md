# Mini Project 1: Website for The Service Benefit

## Client
Deniz Ketenci, Director of The Service Benefit

## Project Description
This website is for The Service Benefit (TSB), a platform that highlights impactful non-governmental organizations (NGOs), promotes volunteer opportunities, and encourages donations to support global causes. The site features a clean, accessible design with pages for the home, about, NGOs, and contact information.

## How to View
The live site is hosted on GitHub Pages at: [https://yourusername.github.io/mini-project-1-website](https://yourusername.github.io/mini-project-1-website)

To view locally:
1. Clone the repository
2. Open `index.html` in a web browser

## What I Learned
- **Semantic HTML**: Used proper semantic elements like `<header>`, `<nav>`, `<main>`, and `<footer>` to improve code readability and accessibility.
- **Navigation**: Implemented consistent, accessible navigation across all pages with clear link structures and hover states.
- **External CSS**: Organized styles in a separate `styles.css` file using CSS variables for maintainability and responsive design.
- **Accessibility**: Focused on accessible design with descriptive alt text for images, keyboard navigation support, and high-contrast color schemes.
